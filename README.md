# Interstellar Beginning

**AI-Smart-Pool**: 面向Web3 AI大模型推理小团体的完全去中心化智能算力池项目。

## 目标
- 去中心化、可扩展、高效的AI推理算力池
- 支持Web3、DAO治理、链上结算、激励、P2P与分布式存储
- 服务AI大模型、小团队、AI应用开发者

## 主要模块
- 节点注册/信誉/KYC
- P2P发现
- 任务调度与结算
- DAO治理
- 插件与扩展
- 分布式存储（IPFS）
- ZK接口预留

## 目录结构
详见 `/src`、`/contracts`、`/scripts`、`/docs` 等文件夹。

## 快速启动
```bash
bash scripts/run_local.sh
```
